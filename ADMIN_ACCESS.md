# 🔐 관리자 접속 가이드

## 숨겨진 관리자 접속 방법

일반 사용자(선수들)에게는 관리자 로그인 버튼이 보이지 않도록 설정했습니다.
관리자만 알고 있는 방법으로 접속할 수 있습니다.

### 📱 접속 방법

**헤더의 로고를 3초 내에 5번 연속 클릭하세요**

1. 웹사이트 상단의 "MIIRACER SPORTS SCORE" 로고를 확인
2. 로고 부분을 **3초 내에** 연속으로 **5번 클릭**
3. 자동으로 관리자 로그인 페이지(/admin/login)로 이동

### ⚡ 동작 원리

- **시간 제한**: 3초 내에 완료해야 함
- **클릭 횟수**: 정확히 5번 클릭
- **초기화**: 3초가 지나면 클릭 카운트가 초기화됨
- **접속**: 5번째 클릭 시 즉시 관리자 로그인 페이지로 이동

### 🎯 장점

- **보안성**: 일반 사용자는 관리자 접속 방법을 모름
- **편의성**: 별도의 URL 암기 불필요
- **직관성**: 로고 클릭만으로 간단한 접속

### 💡 팁

- 모바일에서도 동일하게 작동합니다
- 로고 클릭 시 살짝 투명도가 변하는 호버 효과가 있습니다
- 실패해도 페이지는 새로고침되지 않습니다

### 🚨 비상 접속 방법

만약 위 방법이 작동하지 않을 경우:
```
직접 URL 입력: http://localhost:3000/admin/login
```
